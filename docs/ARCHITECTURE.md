```
┌─────────────────────────────────────────────────────────────┐
│                      总裁（用户）                            │

                              ↑
                    战略/组织/重大决策
                              │
┌─────────────────────────────────────────────────────────────┐
│                     Lysander CEO (AI)                       │

│                                                              │
│  ┌─────────────────────────────────────────────────────┐  │
│  │                   决策体系                            │  │
│  │  decision_check() → 小问题/智囊团/上报                │  │
│  └─────────────────────────────────────────────────────┘  │
│                              │                              │
│  ┌─────────────────────────────────────────────────────┐  │
│  │                   HR知识库                            │  │

                              ↑

                              │

                              │
```

## 数据流

### HR知识库同步

```
OBSidian (人卡) → hr_watcher.py监控 → hr_base.py同步 → *_experts.yaml
                                              ↓
                                     Agent Butler加载
```

### Claude Code记忆

```
OBSidian (记忆) → sync-claude-memory.sh → .claude/ (记忆文件)
                                              ↓
                                     Claude Code加载
```

### 决策流程

```
用户输入 → decision_check() → [判断类型]
                              ↓
           ┌─────────────────┼─────────────────┐
           ↓                 ↓                 ↓
      小问题          需智囊团           超出授权
           ↓                 ↓                 ↓
      直接执行       召集分析后执行      上报总裁
```

## 核心组件

### hr_base.py

HR知识库核心模块：
- `load_org_config()` - 加载组织配置
- `load_personnel_card()` - 加载人员卡片
- `assemble_team_for_task()` - 任务路由
- `decision_check()` - 决策前检查
- `sync_all_teams()` - 同步所有团队

### hr_watcher.py

文件监控自动同步：
- 监控OBS HR目录变化
- 监控Claude Memory变化
- 自动触发同步

### Claude Code Memory

OBSidian中存储的记忆：
- `user_role.md` - 角色定位
- `feedback_*.md` - 经验教训
- `project_*.md` - 项目状态

## 技术栈

| 组件 | 技术 | 用途 |
|------|------|------|
| 知识库 | OBSidian | 单一数据源 |
| AI框架 | CrewAI | 多Agent协作 |
| 自动化 | Python + inotify | 文件监控同步 |
| 决策系统 | hr_base.py | 决策检查 |
| CI/CD | n8n | 博客发布 |
