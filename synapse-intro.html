Synapse 是什么

突触（Synapse）是神经元之间传递信号的关键节点。
我们用同样的逻辑连接三个层：知识、决策、执行。

控制层设计

三类控制机制让 AI 行为可预期、可验证、可演进

Guides · 前馈控制
行为约束系统
定义 AI 的角色锚定、目标对齐、任务分类逻辑。 CEO Guard 硬约束确保执行链不被绕过。 P0 约束优先级高于所有其他规则。
Workflow · 结构化流程
标准执行链 v2.0
开场 → 目标接收 → S/M/L 分级 → 强制派单 → 子 Agent 执行 → QA 审查 → 交付。 每个环节可追溯，不可跳过。
Sensors · 反馈控制
质量门禁
qa_auto_review() 评分 ≥ 4.2/6.0 才能交付。 integration_qa 执行 YAML/代码校验。 execution_auditor 审计执行链完整性。

决策体系

四级决策框架

专业问题留在专家层，只有真正的战略级决策才触达总裁。

L1 · 自动执行
系统自动
决策者：规则引擎
例行操作、标准流程、信息查询。直接执行，记录日志，不打扰任何人。
L2 · 专家评审
智囊团分析
决策者：领域专家
专业问题先由领域专家分析，形成多维建议方案后提交决策。
L3 · 管理决策
Lysander CEO
决策者：AI CEO
基于专家建议做最终管理决策。跨团队协调、资源分配、方案取舍。不上报总裁。
L4 · 总裁决策
总裁刘子杨
决策者：最高决策者
仅限：外部合同签署 · 预算 >100 万 · 公司存续级不可逆决策。

核心原则： 总裁只参与两个阶段——提出目标 和 最终验收。 中间所有过程由 Lysander CEO 全权负责，包括方案设计、决策、执行、审查。
专业的事交给专家，不上报让总裁猜。

Agent HR 管理制度

严格的人才质量标准

每一位 AI 团队成员都需经过审批，能力描述必须达到可验证的方法论级别。

🟢
A 级 · 优秀
"基于 pytest + Playwright 的端到端测试框架搭建与维护"
🟡
B 级 · 合格
"SWOT 分析、PEST 分析、波特五力模型应用"
🔴
C 级 · 自动拒绝
"项目管理"、"知识沉淀"（无方法论，禁止出现）
✓ 新 Agent 须提交 hr_director 入职审批
✓ 重叠度 >30% 不予批准（Jaccard 检测）
✓ 默认 probation 状态，评审通过后升 active
✓ 每周自动运行 audit_all_agents()
自动化编排

无需总裁在线的全天候运转

三个定时 Agent 按 Dubai 时区编排，情报→行动→通知形成完整闭环。
